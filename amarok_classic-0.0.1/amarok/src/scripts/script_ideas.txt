SCRIPT IDEAS
============
This is a list of possible dcop scripts for amaroK. Add your ideas here.


* Alarm
  Starts music at given time.

* Publish
  Generates info about the current track (title, cover...) for a website.

* Playlist2HTML
  Generates a nice HTML chart from the playlist, like Winamp does.

* Speech Announcer
  Announces trackchanges with text-to-speech software. Accessibility aid for the blind.

* Theme Selector
  Manages and installs (downloads?) Context-Browser themes.
